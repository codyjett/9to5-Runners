# 9to5 Runners Website

Static GitHub Pages website for 9to5 Runners.

## Files

- `index.html` — homepage
- `about.html` — about/team page
- `css/` — future shared styles
- `js/` — future scripts
- `images/` — future image assets
- `reviews/` — future shoe review pages
- `brands/` — future brand pages

## GitHub Pages Setup

1. Upload these files to your GitHub repository.
2. Go to **Settings → Pages**.
3. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
4. Save.

Your homepage will load from `index.html`.
