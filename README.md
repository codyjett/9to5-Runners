# 9to5 Runners Full Site

This is the complete self-contained version of the site built so far.

## Main files

- `index.html` — homepage for GitHub Pages
- `about.html` — about/team page
- `9to5_shoes_page1.html` — shoes review page

## Backup/named copies

- `9to5_Homepage.html`
- `9to5_about.html`

## How to upload to GitHub Pages

1. Upload everything in this folder to your repository.
2. Make sure `index.html` is in the root of the repo.
3. Go to GitHub → Settings → Pages.
4. Set source to `Deploy from a branch`.
5. Select your `main` branch and `/root`.
6. Save.

The site should load from `index.html`.
