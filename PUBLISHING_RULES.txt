- If a filter is missing, add it to the existing filter group and wire it to the review row.

ASSET MANAGEMENT
- Never embed images as Base64 in public HTML files.
- Store images in the shared /images directory.
- Reference images with relative paths.
- Reuse shared logos and runner profile graphics.
